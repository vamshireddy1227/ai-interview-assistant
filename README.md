# 🤖 AI Interview Assistant

An AI-powered web application that helps students prepare for HR and technical interviews through interactive questions, voice features, timers, and downloadable reports.

## 🌐 Live Demo

🔗 https://vamshireddy1227.github.io/ai-interview-assistant/

## 💻 GitHub Repository

🔗 https://github.com/vamshireddy1227/ai-interview-assistant

---

## ✨ Features

- 🎤 Voice Input using Web Speech API
- 🔊 Text-to-Speech (Reads Questions)
- ⏱️ 30-Second Timer
- ➡️ Automatic Next Question
- 📚 Multiple Interview Categories
  - HR
  - Python
  - Java
  - C++
  - Technical
  - Data Structures
- 📄 Download Interview Report
- 🏆 Download Certificate
- 🌙 Dark Mode
- 📊 Interview Summary

1. Clone the repository

git clone https://github.com/vamshireddy1227/ai-interview-assistant.git

2. Open the project folder.

3. Open `index.html` in your browser.
